API Documentation

Api Documentation

Base URL: https://kaushalanimesh.com/API/V1/

/auth

  • Register User (POST /signup)
  • Login: (POST /login)

/todo

  • Get all TO-DOs for a user (GET /user/:userid)
  • Add TO-DO (POST /user/:userid)
  • Edit TO-DO (PUT /user/:userid/:todoID)
  • Edit an item in a TO-DO (PUT /user/:userid/:todoID/:itemID)
  • Delete a TO-DO (DELETE /:userid/:todoID)
  • Delete an item in a TO-DO (DELETE /:userid/:todoID/:itemID)

Auth

Register User

Create a new user.

  • URL: POST /signup
  • Sample Request
    {
    "username": "johndoe",
    "password": "asd",
    "firstName": "animesh",
    "lastName":"kaushal"
    }
  • Sample Response
    Status code: 201 – If user Created
    Status code: 409 – If username already taken

Login

Sign in an existing user

  • URL: POST /login
  • Sample Request
    {
    “username”: “animesh”
    “password”: “password”
    }
  • Sample Response
    • Status code: 200
      { “userid” : 2 }
    • Status code: 401

Get all TO-DOs for a user

Get all to-dos and their corresponding to do lists for a signed in user.

  • URL: GET /user/:userid
  • Sample Request
    https://kaushalanimesh.com/API/V1/user/1
  • Sample response
    {"todo": [
    {"todoID": "1", "date": "2002-19-15", "name":"Chore List",
    "todolist": [
    {"itemID" : 5, "description": "Do the laundry"},
    {"itemID" : 6, "description": "Walk the dog"}]
    }]
    }

Add TO-DO

Add a TO-DO for a signed in user.

  • URL: POST /user/:userid
  • Sample Request
    {“todo”: { “name”: “chores for weekend”, “date”: “31-03-2021”,
    “todolist”: [
    {“description”: “pick up groceries”},
    {“description”: “visit grandmother”}
    ]
    }
    }
  • Sample response
    • Status code: 200

Edit TO-DO

Edit a TO-DO for a signed in User.

  • URL: PUT /user/:userid/:todoID
  • Sample Request
    {
    “todoID”: 3,
    “name”: “necessary thing for next week”
    }
  • Sample response
    • Status code: 400 for Bad Request
    • Status code: 204 for Resource Updated Successfully.

Edit an item in a TO-DO

Edit an item from the list of items in a TO-DO for a signed in User.

  • URL: PUT /user/:userid/:todoID/:itemID
  • Sample Request
    {
    “todoID”: 3,
    “itemID”: 59,
    “description”: “Go for Yoga”
    }
  • Sample response
    • Status code: 400 for Bad Request
    • Status code: 204 for Resource Updated Successfully.

Delete a TO-DO

Delete a TO-DO for a signed in User.

  • URL: DELETE /user/:userid/:todoID
  • Sample Request
    {
    “todoID”: 3
    }
  • Sample response
    • Status code: 400 for Bad Request
    • Status code: 204 for Resource Updated Successfully.

Delete an item in a TO-DO

Delete an item from the list of items in a TO-DO for a signed in User.

  • URL: DELETE /user/:userid/:todoID/:itemID
  • Sample Request
    {
    “todoID”: 3,
    “itemID”: 69
    }
  • Sample response
    • Status code: 400 for Bad Request
    • Status code: 204 for Resource Updated Successfully.
